Retro DOS Commands

Command: ASSOC

Description: Displays or modifies file extension associations.

Example: assoc .txt

Command:ATRRIB

Description: Displays or changes file attributes.

Example: ATTRIB attrib +r filename.txt

Command: BREAK

Description: Sets or clears extended CTRL+C checking.

Example: BREAK break on

Command: BCDEDIT

Description: Sets properties in boot database to control boot loading.

Example: BCDEDIT bcdedit /enum

Command: CACLS

Description: Displays or modifies access control lists (ACLs) of files..

Example: CALCS cacls filename.txt

Command: CALL

Description: Calls one batch program from another.

Example: CALL call myscript.bat

Command: CD

Description: Displays the name of or changes the current directory..

Example: CD cd C:\Windows

Command: CHCP

Description: Displays or sets the active code page number.

Example: CHCP chcp 437

Command: CHDIR

Description: Displays the name of or changes the current directory.

Example: CHDIR cd C:\Windows

Command: CHKDSK

Description: Checks a disk and displays a status report..

Example: CHKDSK chkdsk C:

Command: CHKNTFS

Description: Displays or modifies the checking of disk at boot time.

Example: CHKNTFS chkntfs C:

Command: CLS

Description: Clears the screen.

Example: CLS cls

Command: CMD

Description: Starts a new instance of the Windows command interpreter.

Example: CMDcmd

Command: COLOR

Description: Sets the default console foreground and background colors.

Example: COLOR color 0A

Command: COMP

Description: Compares the contents of two files or sets of files.

Example: COMP comp file1.txt file2.txt

Command: COMPACT

Description: Displays or alters the compression of files on NTFS partitions.

Example: COMPACT compact

Command: CONVERT

Description: Converts FAT volumes to NTFS. You cannot convert the current drive.

Example: CONVERT convert C: /fs:ntfs

Command: COPY

Description: Copies one or more files to another location.

Example: COPY copy file1.txt backup\

Command: DATE

Description: Displays or sets the date.

Example: DATE date

Command: DEL

Description:Deletes one or more files.

Example: DEL del oldfile.txt

Command: DIR

Description: Displays a list of files and subdirectories in a directory.

Example: DIR dir

Command: DISKPART

Description: Displays or configures Disk Partition properties.

Example: DISKPART diskpart list disk

Command: DOSKEY

Description: Edits command lines, recalls Windows commands, and creates macros.

Example: DOSKEY doskey ls=dir

Command: DRIVERQUERY

Description: Displays current device driver status and properties.

Example: DRIVERQUERY driverquery

Command: ECHO

Description: Displays messages, or turns command echoing on or off.

Example: ECHO echo Hello, World!

Command: ENDLOCAL

Description: Ends localization of environment changes in a batch file.

Example: ENDLOCALc: \>endlocal

Command: ERASE

Description: Deletes one or more files.

Example: ERASE DEL (ERASE) [d:][path]filename [/P]

Command: EXIT

Description: Quits the CMD.EXE program (command interpreter).

Example: EXIT exit

Command: FC

Description: Compares two files or sets of files, and displays the differences between them.

Example: FC fc file1.txt file2.txt

Command: FIND

Description:Searches for a text string in a file or files.

Example: FIND find "keyword" file.txt

Command: FINDSTR

Description: Searches for strings in files.

Example: FINDSTR findstr "pattern" file.txt

Command: FOR

Description: Runs a specified command for each file in a set of files.

Example: FOR for %f in (*) do echo %f

Command: FORMAT

Description: Formats a disk for use with Windows.

Example: FORMAT format D: /fs:ntfs

Command: FSUTIL

Description: Displays or configures the file system properties.

Example: FSUTIL fsutil fsinfo volumeinfo C:

Command: FTYPE

Description: Displays or modifies file types used in file extension associations.

Example: FTYPE ftype txtfile

Command: GOTO

Description: Directs the Windows command interpreter to a labeled line in a batch program.

Example: GOTO c: \>goto

Command: GPRESULT

Description: Displays Group Policy information for machine or user.

Example: GPRESULT gpresult /r

Command: GRAFTABL

Description: Enables Windows to display an extended character set in graphics mode.

Example: GRAFTABL No example provided as this command is rarely used in modern Windows systems.

Command: HELP

Description: Provides Help information for Windows commands.

Example: HELP help copy

Command: ICACLS

Description: Display, modify, backup, or restore ACLs for files and directories.

Example: ICACLS icacls filename.txt

Command: IF

Description: Performs conditional processing in batch programs.

Example: IF Used in batch files, so there's no single example.

Command: LABEL

Description: Creates, changes, or deletes the volume label of a disk.

Example: LABELlabel C: Data

Command: MD

Description: Creates a directory.

Example: MD md docs

Command: MKDIR

Description: Creates a directory.

Example: MKDIR mkdir docs

Command: MKLINK

Description: Creates Symbolic Links and Hard Links

Example: MKLINK mklink /d D:\file.txt C:\file.txt

Command: MODE

Description: Configures a system device.

Example: MODE No example provided as this command is rarely used in modern Windows systems.

Command: MORE

Description:Displays output one screen at a time.

Example: MORE To display the contents of file.txt one screen at a time, you can use:

Command: MOVE

Description: Moves one or more files from one directory to another directory.

Example: MOVE move file.txt backup\

Command: OPENFILES

Description: Displays files opened by remote users for a file share.

Example: OPENFILES openfiles /query /s Server

Command: PATH

Description: Displays or sets a search path for executable files.

Example: PATH path

Command: PAUSE

Description: Suspends processing of a batch file and displays a message.

Example: PAUSE pause

Command: POPD

Description: Restores the previous value of the current directory saved by PUSHD.

Example: POPD Used in batch files, so there's no single example.

Command: PRINT

Description: Prints a text file.

Example: PRINT print file.txt

Command: PROMPT

Description: Changes the Windows command prompt.

Example: PROMPT prompt $P$G

Command: PUSHD

Description: Saves the current directory then changes it.

Example: PUSHD PUSHD [drive]path PUSHD

Command: RD

Description: Removes a directory.

Example: RD rd docs

Command: RECOVER

Description: Recovers readable information from a bad or defective disk.

Example: RECOVER recover file.txt

Command: REM

Description: Records comments (remarks) in batch files or CONFIG.SYS.

Example: REM REM [comment]

Command: REN

Description: Renames a file or files.

Example: REN ren oldfile.txt newfile.txt

Command: RENAME

Description: Renames a file or files.

Example: RENAME ren oldfile.txt newfile.txt

Command: REPLACE

Description: Replaces files.

Example: REPLACE replace file1.txt file2.txt

Command: RMDIR

Description: Removes a directory.

Example: RMDIR rmdir directoryname

Command: ROBOCOPY

Description: Advanced utility to copy files and directory trees.

Example: ROBOCOPY robocopy sourcedir destinationdir /E

Command: SET

Description: Displays, sets, or removes Windows environment variables.

Example: SET set PATH=C:\NewPath;%PATH%

Command: SETLOCAL

Description: Begins localization of environment changes in a batch file.

Example: SETLOCAL setlocal

Command: SC

Description: Displays or configures services (background processes).

Example: SC sc query

Command: SCHTASKS

Description: Schedules commands and programs to run on a computer.

Example: SCHTASKS schtasks /create /sc daily /tn "MyTask" /tr "C:\Path\to\MyProgram.exe"

Command: SHIFT

Description: Shifts the position of replaceable parameters in batch files.

Example: SHIFT shift

Command: SHUTDOWN

Description: Allows proper local or remote shutdown of machine.

Example: SHUTDOWN shutdown /s /t 0

Command: SORT

Description: Sorts input.

Example: SORT sort file.txt

Command: START

Description: Starts a separate window to run a specified program or command.

Example: START start notepad.exe

Command: SUBST

Description: Associates a path with a drive letter.

Example: SUBST subst X: C:\Path\to\Directory

Command: SYSTEMINFO

Description: Displays machine specific properties and configuration.

Example: SYSTEMINFO systeminfo

Command: TASKLIST

Description: Displays all currently running tasks including services.

Example: TASKLIST tasklist

Command: TASKKILL

Description: Kill or stop a running process or application.

Example: TASKKILL taskkill /im notepad.exe

Command: TIME

Description: Displays or sets the system time.

Example: TIME time

Command: TITLE

Description: Sets the window title for a CMD.EXE session.

Example: TITLE title My Command Prompt

Command: TREE

Description: Graphically displays the directory structure of a drive or path.

Example: TREE tree /f

Command: TYPE

Description: Displays the contents of a text file.

Example: TYPE type file.txt

Command: VER

Description: Displays the Windows version.

Example: VER ver

Command: VERIFY

Description: Tells Windows whether to verify that your files are written correctly to a disk.

Example: VERIFY verify on

Command: VOL

Description: Displays a disk volume label and serial number.

Example: VOL vol C:

Command: XCOPY

Description: Copies files and directory trees.

Example: XCOPY xcopy sourcedir destinationdir /E

Command: WMIC

Description: Displays WMI information inside interactive command shell.

Example: WMIC wmic
